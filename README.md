# Solar Energy Production Forecasting

Machine learning project focused on predicting solar energy production using weather data, feature engineering and advanced ensemble models.

## Project Overview

The objective of this project is to forecast short-term solar energy production and outperform an industrial benchmark using machine learning techniques.

The project includes:
- Data exploration
- Feature engineering
- Model selection and tuning
- Explainable AI (XAI)
- Performance benchmarking

---

## Features

- Time series forecasting
- Feature engineering for temporal data
- Chronological train/validation split
- Multiple ML models comparison
- XGBoost optimization
- SHAP explainability analysis
- Permutation importance interpretation

---

## Models Tested

- Ridge Regression
- Lasso Regression
- K-Nearest Neighbors (KNN)
- Random Forest
- XGBoost
- LightGBM

---

## Final Results

### Selected Model: XGBoost

- RMSE: 0.0029
- R² Score: 0.95
- ~55% RMSE improvement over benchmark
